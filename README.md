# 🔑 PassOP – Secure Credential Storage System

PassOP is a **password management application** designed to streamline credential storage and enhance security.  
Built using the **MERN stack**, it allows users to securely store, edit, and manage their credentials.

---

## 🚀 Features
- 📱 **Responsive Design** – Built with **React** and **Material-UI**.
- ✏️ **Edit & Delete Credentials** – Manage saved passwords with ease.
- 🗄️ **Secure Storage** – Passwords stored in **MongoDB** for scalability.
- 📦 **Dependency Management** – Handled via **npm** for smooth builds.

---

## 🛠️ Tech Stack
- **Frontend:** React, Material-UI
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **State Management:** React Hooks
